# import-cleanup-tool-backend

## Get started

<ol>
<li>Run `npm i`</li>
<li>Create a .env file and include the following variables:
    <ul>
        <li>MANDRILL_API_KEY</li>
        <li>HERE_API_KEY</li>
        <li>PORT (optional)</li>
    </ul>
</li>
<li>Run `node .`</li>
<li>Open up http://localhost:{PORT} in your browser</li>
</ol>





Node version used: v20.9.0
